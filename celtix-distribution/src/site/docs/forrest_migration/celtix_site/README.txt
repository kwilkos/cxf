This is the top level of the Forrest project to build the CeliXfire documentation and Web site.
To build the site you need the following:
1. Apache Forrest v0.7
2. The latest version of the Docbook XSLT stylesheets.

Before building the site you need to do the following:
1. Install Apache Forrest.
2. Copy the celtix-site skin, availble from the repository, to the
   main/webapp/skins folder of your Forrest installation.
3. Copy the Docbook XSLT stylesheets to the 
   src\documentation\resources\stylesheets\docbook folder under this
   directory.
   
To build the site type forrest from the prompt while in this directory.
